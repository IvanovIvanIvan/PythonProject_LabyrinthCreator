Генератор лабиринтов.
Данный проект способен генерировать лабиринты различных вариантов генерации, сохранять/загружать их из/в файлов, выводить их в консоль и файлы, а также решать лабиринты и отображать прогресс в них.


Код умеет создавать неориентированный граф, соответствующий лабиринту и хранит его как матрицу связности. К графу предъявлены следующие требования: 
* рёбра в нём окрашены в 4 цвета
* из каждой вершины выходит не более одного ребра каждого цвета
* в нём существует путь из вершины с номером 0 к вершине с номером (V-1), где V - число вершин в графе.
Граф генерируется с помощью алгоритма DFS или минимального остовного дерева.
При выводе граф преобразуется в 2D-лабиринт. При вводе введённый лабиринт преобразуется в граф.
Также есть возможность решить лабиринт и вывести на экран правильный путь прохождения.


Архитектура:
Класс LabrinthGraph - граф, на котором основывается лабиринт, поддерживающий методы:
random_construct(V, alg) - конструирование случайного графа с числом вершин V, удовлетворяющего требованиям выше, с использованием алгоритма alg.
solve() - решает лабиринт и возвращает путь выхода из лабиринта.
Оператор [], позволяющий пользователю самостоятельно создать лабиринт.


От него унаследуем класс Labrinth
В нём дополнительно реализуем методы:
print() - выводит лабиринт в консоль
printf(dest) - записывает лабиринт в файл, находящийся по адресу dest
readf(dest) - считывает лабиринт из файла по адресу dest
